# BloodDonor

BloodDonor is an Android application project for connecting blood donors with those in need.

## Features
- Search and view donor information
- User-friendly Android interface
- Built with Kotlin and Android Studio

## Getting Started
1. Open the project in Android Studio.
2. Sync Gradle files.
3. Run the app on an emulator or physical device.

## Notes
- Local configuration files such as local.properties are ignored by git.
- Sensitive signing files are also ignored by git.
